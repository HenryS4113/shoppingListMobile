# Shopping List App

This program allows users to add items, set quantity and priority levels, and sort through an infinitly expanding list of grocery and non grocery items.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
